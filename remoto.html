2.- Conexión de escritorio remoto en Windows (RDP)

Levantar el servicio RDP en windows

Mi manera de entender este enunciado es levantar en un Windows Server el servicio RDP y luego establecer una conexión RDP desde un cliente Windows y Linux (usando Remmina), así que para esto lo que haremos será levantar el servicio RDP en Windows Server de esta manera:

Abrimos configuración de Windows y vamos al apartado “Sistema”.

Una vez aquí vamos a escritorio remoto y lo habilitamos.

Y antes de ir a la parte cliente le cambiaremos el nombre al servidor por “NombreApellido” tal como se indicaba en los enunciados anteriores.

Conexión cliente RDP desde:

  • Windows

Aquí en el cliente es muy importante que activemos en escritorio remoto “Permitir las conexiones remotas a este equipo” y aplicamos y aceptamos.

Y ahora desde el cliente usaremos conexión remota para establecer conexión RDP con el servidor.

Además tendremos la casilla de “Recordar credenciales” por si queremos establecer mas conexiones recurrentes a ese equipo y no tener que introducir nombre y contraseña.

Cabe destacar que como el perfil creado es Administrador y no pude cambiarlo porque se necesita activar Windows para ello se tiene que quedar así, pero como cambié el nombre del equipo en este aviso siguiente se puede ver correctamente el nombre, ya que este es un aviso para confirmar si quieres establecer la conexión.
